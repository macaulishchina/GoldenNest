# 主题系统使用说明

## 功能概述

小金库应用现在支持多种主题，用户可以根据个人喜好选择不同的视觉风格。

## 可用主题

### 1. ☀️ 浅色主题 (Light)
- **特点**: 清新明亮的浅色界面
- **主色调**: 翠绿色 (#18A058)
- **适用场景**: 日间使用，光线充足的环境

### 2. 🌙 深色主题 (Dark)
- **特点**: 护眼舒适的深色界面
- **主色调**: 薄荷绿 (#63E2B7)
- **适用场景**: 夜间使用，减少眼睛疲劳

### 3. 🌊 海洋蓝 (Blue)
- **特点**: 清新海洋风格
- **主色调**: 天空蓝 (#2080F0)
- **背景**: 淡蓝色调 (#F0F7FF)
- **适用场景**: 喜欢蓝色调的用户

### 4. 🌲 森林绿 (Green)
- **特点**: 自然清新风格
- **主色调**: 生机绿 (#52C41A)
- **背景**: 淡绿色调 (#F6FFED)
- **适用场景**: 追求自然感觉的用户

### 5. 🔮 神秘紫 (Purple)
- **特点**: 优雅神秘风格
- **主色调**: 典雅紫 (#722ED1)
- **背景**: 淡紫色调 (#F9F0FF)
- **适用场景**: 喜欢优雅风格的用户

## 使用方法

### 切换主题

1. **桌面端**:
   - 点击页面右上角的主题图标按钮
   - 从弹出的主题选择器中选择你喜欢的主题

2. **移动端**:
   - 点击页面右上角的主题图标按钮（汉堡菜单左侧）
   - 从弹出的主题选择器中选择你喜欢的主题

3. **登录页面**:
   - 在登录页面右上角也可以切换主题
   - 可以在登录前预览不同主题效果

### 主题持久化

- 用户选择的主题会自动保存在浏览器本地存储中
- 下次访问时会自动应用上次选择的主题
- 不需要重新登录或刷新页面

## 技术实现

### 架构设计

```
frontend/src/
├── stores/
│   └── theme.ts          # 主题状态管理 (Pinia Store)
├── components/
│   └── ThemeSelector.vue # 主题选择器组件
├── App.vue               # 应用根组件（配置主题提供者）
└── assets/
    └── main.css          # 全局样式（主题背景色）
```

### 核心功能

1. **主题Store** (`stores/theme.ts`)
   - 管理当前主题状态
   - 提供主题配置和覆盖
   - 实现主题持久化

2. **主题选择器** (`components/ThemeSelector.vue`)
   - 可视化主题选择界面
   - 显示主题图标、名称和描述
   - 响应式设计，适配移动端

3. **主题配置**
   - 基于 Naive UI 的主题系统
   - 支持明暗模式切换
   - 自定义颜色方案

### 扩展主题

如需添加新主题，修改 `stores/theme.ts`:

```typescript
// 1. 在主题列表中添加新主题
const themes = [
  // ... 现有主题
  {
    name: 'orange',
    label: '活力橙',
    icon: '🍊',
    description: '充满活力的橙色主题'
  }
]

// 2. 在 themeOverrides 中添加配置
case 'orange':
  return {
    ...commonOverrides,
    common: {
      primaryColor: '#FF6B00',
      // ... 其他颜色配置
    }
  }
```

## 最佳实践

1. **选择建议**:
   - 白天使用浅色主题或彩色主题
   - 夜间使用深色主题保护视力
   - 根据个人喜好选择彩色主题

2. **性能优化**:
   - 主题切换使用 CSS 过渡动画，流畅自然
   - 主题配置存储在本地，无需网络请求
   - 支持即时切换，无需刷新页面

3. **用户体验**:
   - 主题选择器提供清晰的预览
   - 图标和描述帮助用户快速识别
   - 当前主题有明显的视觉标识
